## 内容介绍
Sakura是一个Minecraft Hacked客户端，允许开发者对游戏源代码进行修改和重新编译。主要功能包括：
- 反编译Minecraft源代码
- 黑客功能的编写和使用
- 提供Gradle构建系统支持
- 生成可运行的修改版客户端
- 支持Optifine集成
- 集成的游戏优化

## 布吉岛
《布吉岛》是一款充满挑战与策略的竞技类 HvH（黑客对黑客）游戏，玩家需要运用各种技巧和策略来击败对手。在这个虚拟的世界中，玩家将扮演一名勇敢的战士，与敌人展开激烈的对抗。为了在战斗中占据优势，玩家将会选择使用如 KillAura 和 Speed 等黑客功能，以提高自己的战斗力
游戏的目标是消灭所有绿色阵营的玩家。这些绿色玩家不仅是你的直接竞争对手，更是你通往胜利的障碍。每一场战斗都充满了紧张与刺激，你需要时刻保持警惕，利用手中的武器和技能，迅速而精准地击溃敌人。只有当你成功消灭所有绿色玩家后，才能迎接最终的挑战——阻止圈钱祖师爷“妖猫”开上宝马
妖猫是游戏中的最终BOSS，他不仅拥有强大的力量，还具备高超的圈钱意识。要阻止他开上宝马并非易事，这需要团队的默契配合和个人的卓越表现。一旦妖猫成功开上宝马，游戏即宣告失败。更糟糕的是，失败的玩家将会被妖猫使用四大反作弊系统囚禁在小黑屋，失去再次挑战的机会
为了取得胜利，玩家必须精心策划每一个行动，合理分配资源，并与队友紧密合作。每一次决策都至关重要，每一个瞬间都可能决定成败。在这个充满未知与挑战的布吉岛上，只有最聪明、最勇敢的玩家才能笑到最后，成为真正的胜利者
